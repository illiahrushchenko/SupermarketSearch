# SupermarketSearch Mockup

Це демонстраційний **HTML/CSS мокап** багатосторінкового веб-додатку для аналізу й порівняння цін у супермаркетах.

## Сторінки
- `index.html` — головна сторінка з пошуком, фільтрами, сортуванням, таблицею порівняння цін та статистикою (min/avg/max).
- `favorites.html` — розділ «Вподобані» (wishlist).
- `account.html` — розділ «Кабінет» (реєстрація/автентифікація, мокап).
- `chat.html` — сторінка AI-чату (демо-відповіді).

## Запуск
Оскільки це статичний мокап, просто відкрийте `index.html` у браузері.

Альтернатива (через локальний сервер):
```bash
python3 -m http.server 8080
```
Після цього відкрийте `http://localhost:8080/index.html`.

## Інструкції тестування основних флоу
1. **Навігація між сторінками**
   - Відкрийте `index.html`.
   - Перейдіть через меню у «Вподобані», «Кабінет», «AI-чат».
   - Переконайтеся, що активний пункт меню підсвічується на кожній сторінці.

2. **Пошук, фільтрація, сортування (демо-інтерфейс)**
   - На `index.html` перевірте наявність поля пошуку, випадаючого каталогу, сортування.
   - Зафіксуйте скріншот екрану як демонстрацію робочого UX-флоу.

3. **Порівняння цін і аналітика**
   - На `index.html` перевірте таблицю товарів з цінами по мережах.
   - Перевірте відображення карток: мінімальна / середня / максимальна ціна.

4. **Wishlist**
   - Перейдіть у `favorites.html`.
   - Перевірте список вподобаних товарів і кнопки швидкого переходу.

5. **Кабінет користувача**
   - Перейдіть у `account.html`.
   - Перевірте наявність полів реєстрації/входу та кнопок переходу до wishlist.

6. **AI-чат**
   - Перейдіть у `chat.html`.
   - Перевірте історію повідомлень, поле нового запиту й кнопки дій.

## Важливо
Це **мокап для скріншотів і демонстрації UX**, без реальної БД, бекенду, автентифікації чи AI-інтеграції.
